# Landing Page Ingresso.com
> Desafio para a vaga de Front-end Developer, que consiste no desenvolvimento de uma aplicação react de acordo com o layout proposto.

## Instalação

Instalando o Back-end:

```sh
cd server
npm install
npm run server
```

Instalando o Front-end:

```sh
cd client
npm install
npm start
```
